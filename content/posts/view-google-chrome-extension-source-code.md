- Go to `chrome://extensions/`
- Find extension and copy ID
- Go to `%localappdata%`
- `C:\\Users\\{username}\\AppData\\Local\\Google\\Chrome\\User Data\\Default\\Extensions`
- Find extension folder
- Bingo!
